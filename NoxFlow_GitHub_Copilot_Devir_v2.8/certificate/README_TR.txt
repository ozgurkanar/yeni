PAKET İÇİ CHARLES KÖK SERTİFİKASI

Bu klasördeki downloadfile.crt kullanıcı tarafından sağlanan Charles kök sertifikasıdır.
Runtime sertifikayı dışarıdan aramaz ve eski sürüm klasörlerine bağlanmaz.

Windows kaynağı (göreli):
certificate\downloadfile.crt

Nox hedefi:
/sdcard/Download/downloadfile.crt

Yeni bir Charles kök sertifikası kullanmanız gerekirse yalnızca bu klasördeki
downloadfile.crt dosyasını aynı adla değiştirin.
