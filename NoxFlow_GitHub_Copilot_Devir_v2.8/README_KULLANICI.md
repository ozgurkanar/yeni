# NoxFlow Kullanıcı Kılavuzu

1. ZIP'i yazma izni olan normal bir klasöre tamamen çıkarın.
2. NoxPlayer, Charles Proxy ve NoxFlow kapalıyken `NoxFlow.exe`yi yönetici olarak çalıştırın.
3. **Başlat** düğmesine basın.
4. Program tüm Nox görevlerini temizler, NoxConsole ile yeni `Nox_1` klonu üretir, açar, Charles/proxy ve sertifikayı hazırlar, ardından gömülü akışı çalıştırır.
5. **Durdur** veya pencerenin X düğmesi güvenli kapanışı başlatır.

## Gerekli kurulumlar

- NoxPlayer: varsayılan yol `C:\Program Files (x86)\Nox\bin`
- NoxConsole.exe ve nox_adb.exe erişilebilir olmalı.
- Kaynak sanal makine adı `nox` olmalı.
- Kaynak `nox` üzerinde High FPS Mode 120 önceden açık olmalı.
- Charles Proxy kurulmuş olmalı.

Sorun halinde Günlük sekmesindeki son 30-50 satırı paylaşın.
